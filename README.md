# Grocery Store SQL Data Analysis

## Project Overview
This project analyzes grocery store operations using SQL.

The database includes information about:
- Customers
- Products
- Orders
- Employees
- Suppliers
- Categories

## Key Analysis

### Customer Insights
- Unique customers
- Top customers by purchase value

### Product Performance
- Best selling products
- Category performance

### Sales Trends
- Monthly revenue
- Order trends

### Supplier Analysis
- Supplier contribution to revenue

## Tools Used
- MySQL
- SQL Joins
- Aggregations
- Window Functions
